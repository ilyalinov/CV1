### Overlap detection app for ADAS

[![Build Status](https://travis-ci.com/ilyalinov/CV1.svg?branch=dev)](https://travis-ci.com/ilyalinov/CV1)

#### Prerequisites:
* GCC or Visual C++ compiler
* OpenCV
* CMake

#### How to build:
* `cd overlapDetection`
* `mkdir build && cd build`
* `cmake ..`
* `cmake --build .`

#### How to run:
* `overlapDetection <path_to_video>`
